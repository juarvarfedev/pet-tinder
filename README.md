# Pet Rating App

A lightweight browser-based pet swipe app vibe-coded with GitHub Copilot. Users can like or pass on random pets, undo their last decision, and keep browsing through a continuously loaded deck.

## Features

- Swipe-style card interface for rating pets
- Like / pass actions
- Undo support for the previous rating
- Keyboard shortcuts:
  - L = Like
  - P = Pass
  - U = Undo
- Drag-and-drop interaction on the pet card
- Automatic loading of additional pets from a public API

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Files

- `index.html` — app structure and layout
- `styles.css` — visual styling and card animation
- `script.js` — app logic, API fetching, and interactions

## Run Locally

This project is a static web app, so no install step is required.

1. Open `index.html` in your browser, or
2. Use a local preview extension such as Live Server in VS Code.

## How to Use

- Click the Like or Pass buttons
- Drag the pet card left or right
- Press the keyboard shortcuts shown in the footer
